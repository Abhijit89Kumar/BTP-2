## 3. Triton Kernel — 2-D Grid Layout & Tiled Execution

```mermaid
flowchart TB
    subgraph Grid["2-D Grid: ceil(N/BM) x ceil(S/BN)"]
        direction TB
        PID["Each program instance pid(m, n)"]

        subgraph KLOOP["K-Loop (in SRAM)"]
            L1["tl.load L_tile [BM x BK]"]
            L2["tl.load Z_tile [BK x BN]"]
            L3["acc += tl.dot(L_tile, Z_tile)"]
            L1 --> L3
            L2 --> L3
        end

        subgraph FUSE["Fused Business Logic (in SRAM)"]
            F1["D = mu + acc"]
            F2["D = max(D, 0)"]
            F3["X = min(D, Q)"]
            F4["profit = p*X - c*Q + s*max(Q-D, 0)"]
            F5["partial_mean = sum(profit) / S"]
            F1 --> F2 --> F3 --> F4 --> F5
        end

        PID --> KLOOP
        KLOOP --> FUSE
    end

    FUSE --> OUT["tl.atomic_add → out[N]<br/>(ONLY HBM write)"]

    style Grid fill:#FFF8E1,stroke:#FF6F00,stroke-width:2px
    style KLOOP fill:#E3F2FD,stroke:#1565C0
    style FUSE fill:#FCE4EC,stroke:#D32F2F
    style OUT fill:#C8E6C9,stroke:#2E7D32,stroke-width:3px
```

## 4. Memory Hierarchy — PyTorch Baseline vs Triton Fused

```mermaid
flowchart TB
    subgraph LEFT["PyTorch + torch.compile"]
        direction TB
        P1["L, Z loaded from HBM"]
        P2["D = mu + L @ Z<br/>⚠ D [N x S] WRITTEN TO HBM<br/>2048 x 131072 x 4B = 1.07 GB"]
        P3["D RE-READ from HBM<br/>X = min(D, Q)"]
        P4["Profit WRITTEN to HBM<br/>then reduced"]
        P5["❌ 3 full N x S HBM round-trips<br/>~3.2 GB traffic"]
        P1 --> P2 --> P3 --> P4 --> P5
    end

    subgraph RIGHT["Triton Fused Kernel"]
        direction TB
        T1["tl.load L_tile, Z_tile → SRAM"]
        T2["acc = tl.dot in registers"]
        T3["D, X, Profit — ALL IN SRAM"]
        T4["partial_mean — SRAM reduction"]
        T5["✅ atomic_add → out[N]<br/>Only 8 KB for N=2048<br/>D [N x S] NEVER materialised"]
        T1 --> T2 --> T3 --> T4 --> T5
    end

    style LEFT fill:#FFCDD2,stroke:#D32F2F,stroke-width:2px
    style RIGHT fill:#C8E6C9,stroke:#2E7D32,stroke-width:2px
    style P2 fill:#D32F2F,color:#fff
    style P5 fill:#FFCDD2,stroke:#D32F2F
    style T5 fill:#A5D6A7,stroke:#2E7D32
```
